# モンスターたんけん（ポケットモンスター赤緑のパクリゲー）

草むらを歩いてモンスターと出会い、ボールで捕まえて仲間を増やすRPG。

- 最初に「ほのお／みず／くさ」から相棒モンスターを1匹えらぶ。
- 矢印キー（スマホは画面下のジョイパッド）で町とフィールドを歩く。
- 草むら（こい緑）に入るとランダムでモンスターが現れバトル開始。
- バトルは「たたかう／ボール／いれかえ／にげる」のコマンド制。タイプ相性（ほのお＞くさ＞みず＞ほのお）あり。
- 勝つと経験値が入りレベルアップ。一定レベルで進化する。
- ボールで弱った野生モンスターを捕まえて手持ち（最大6匹）を増やす。
- 図鑑（つかまえた種類）はブラウザに保存される。

商標回避のためモンスター名・技名はすべてオリジナル。全部1ファイル（＋core.js）で動くブラウザゲーム。
