# 原神風 宝石ハンター

原神のパクリゲー10人ぐらい出てくる
アイテムは6個持てる(キラと水晶を合わせて持つ)
キラは16個まで持てる(重ねて持てる)
キラは1個で1回しか使えない
キラは水晶を使うためにある

水晶は4種類:
1. 誰が何を持っているか見える
2. 攻撃
3. 1つ目(攻撃)の強化版、音も聞こえる、アイテムが何かもわかる
4. 回復+攻撃

10人のキャラは付き替え可能
キャラ同士で何かを探し合う
誰か1人を操作する
視点はキャラ目線
探しているもの(宝石・星のかけら)を10個以上集めたらクリア
マップ上と他のキャラが持っているのと両方ある
他の人は攻撃しない、ヒントを教えてくれる
コインは集めない
スマホタッチ操作対応
