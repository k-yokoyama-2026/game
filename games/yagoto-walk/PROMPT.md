# 八事ウォーク（位置ゲー）

愛知県名古屋市の八事日赤駅付近を舞台にした位置情報ゲーム。
チェックポイント（八事日赤駅・八事日赤病院・興正寺・中京大学・隼人池・八事駅・平和公園 など）を
実際に歩いてめぐると、到達するたびにポイントが貯まる。

- ブラウザの位置情報（Geolocation API / HTTPS）で現在地を取得
- チェックポイント半径45m以内に入ると到達＝ポイント獲得
- 短時間で次々めぐると「れんさ」ボーナス、全制覇でボーナス
- Leaflet + OpenStreetMap で地図・ピン・現在地を表示
- 現地に行けない人向けの「テストモード」（地図タップ／リストの「行く」で擬似移動）

※ チェックポイントの緯度経度は暫定。`CHECKPOINTS` 配列で調整できる。
